# Bank Loan Analysis Dashboard Project

## Project Overview

This project involves the development of a comprehensive **Bank Loan Analysis Dashboard** using Power BI, Excel, and MySQL. The dashboard is designed to provide HR and business analysts with actionable insights into the bank’s loan portfolio, enabling informed, data-driven decisions.

The project focuses on visualizing key metrics, trends, and patterns in loan applications, funding, interest rates, and debt-to-income ratios across multiple dimensions.

---

## Repository Contents

* **datasets/** – Contains all CSV datasets used for the dashboard.
* **sql_queries/** – SQL scripts used to extract, clean, and prepare data for analysis.
* **dashboards/** – PNG images of dashboard pages showing visualizations and insights.

---

## Dashboard Pages

### 1. Summary Page

Provides a high-level overview of the bank's loan performance with key metrics:

* **Total Loan Applications:** 38.6K
* **Total Amount Received:** $473.1M
* **Total Amount Funded:** $435.8M
* **Average Interest Rate:** 12.05%
* **Average Debt-to-Income (DTI):** 13.3%

Breakdown of loan quality:

* **Good Loans:**

  * Applications: 33.2K (86.2%)
  * Funded Amount: $370.2M
  * Received Amount: $435.8M

* **Bad Loans:**

  * Applications: 5.3K (13.8%)
  * Funded Amount: $65.5M
  * Received Amount: $37.3M

Also includes **Loan Status** metrics for current, charged-off, and fully paid loans.

---

### 2. Overview Page

Visualizes trends across multiple dimensions:

* **Monthly Loan Applications:** Line chart showing trends over time.
* **Applications by State:** Geographical map highlighting application distribution.
* **Applications by Employee Length:** Bar chart for tenure-based distribution.
* **Applications by Loan Purpose:** Debt consolidation, home improvement, major purchases, etc.
* **Applications by Term:** Pie chart showing 36- vs 60-month loans.
* **Applications by Home Ownership:** Comparative analysis by mortgage, rent, and ownership.

---

### 3. Details Page

Displays all dataset records in list view for reference and validation.
